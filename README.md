# 🎓 Udemy Clone

A simple and responsive **Udemy Clone** built using **HTML5** and **CSS3**. This project replicates the basic layout and design of the Udemy homepage, showcasing courses, categories, and a modern user interface.

## 🚀 Features

* Responsive Navigation Bar
* Hero Section
* Course Categories
* Recommended Courses
* Popular Courses Section
* Topics Section
* Footer Section
* Mobile-Friendly Design

## 🛠️ Built With

* HTML5
* CSS3
